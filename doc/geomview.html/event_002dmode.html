7.2.51 event-mode

(event-mode MODESTRING)
Set the mouse event (motion) mode; MODESTRING should be one of the following strings:
  1. "[r] Rotate"
  2. "[t] Translate"
  3. "[z] Cam Zoom"
  4. "[s] Geom Scale"
  5. "[f] Cam Fly"
  6. "[o] Cam Orbit"
  7. "[le] Edit Lights"
